# Contributing

Thanks for your interest in contributing!

## Getting Started

1. Install dependencies: `pnpm install --recursive`
2. Run the dev servers: `pnpm dev`
3. Run checks: `pnpm lint && pnpm type-check && pnpm build`

## Pull Requests

- Keep changes focused and small.
- Add or update tests when applicable.
- Ensure lint and type-check pass.

## Code Style

- Prefer explicit types over `any`.
- Keep files organized and avoid duplicate configs.

## Reporting Issues

Please include:
- Steps to reproduce
- Expected vs actual behavior
- Logs or screenshots if relevant
