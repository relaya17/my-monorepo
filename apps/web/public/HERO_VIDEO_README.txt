Hero background video (Landing page)
====================================

To enable the cinematic ambient video in the Hero section:

1. Add a file named: hero-bg.webm

2. Format: .webm (lightweight, fast). Place in this folder (apps/web/public/).

3. The page uses 60% black overlay over the video so white text stays readable.
   If the file is missing, the Hero still shows a dark background.
