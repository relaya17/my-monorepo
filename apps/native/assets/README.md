# App Assets

Place the following files here before building:

| File | Size | Description |
|------|------|-------------|
| `icon.png` | 1024×1024 px | App icon (iOS & Android) |
| `adaptive-icon.png` | 1024×1024 px | Android adaptive icon foreground |
| `splash.png` | 1284×2778 px | Splash screen image |
| `favicon.png` | 48×48 px | Web favicon |
| `notification-icon.png` | 96×96 px | Android notification icon (white on transparent) |

**Design tokens:**
- Background color: `#0A0E17`
- Primary brand color: `#1E3A8A`

Generate all sizes automatically with `npx expo-optimize` or the [Expo icon tools](https://docs.expo.dev/develop/user-interface/splash-screen-and-app-icon/).
